## Summary

`v0.1.14` published the OpenAI API changelog source snapshot for install-only
users. The bundled reviewed feed now has forty-five ProviderEvents and latest
event date `2026-06-11`.

This release includes the deterministic `openai.api_changelog` source and the
reviewed OpenAI API events for workload identity federation, `chat-latest`,
prompt-cache retention, container-session billing, and moderation-score API
responses. It does not move or recreate the signed `data-2026.06.11` immutable
data tag. It does not change event-promotion authority, source-owner authority,
release-token boundaries, or the read-only MCP default.

## Known Caveats

- The PyPI exact-version JSON exposed `0.1.14` before the HTML Simple API
  response used by resolvers included the new files. The normal resolver
  install by exact version passed after the Simple API cache refreshed.
- `v0.1.14` is a read-only package snapshot for the latest reviewed public
  bundled feed. It does not permit unattended event publication and does not
  change the policy that generated candidates require source-owner review
  before promotion.
