## Known Caveats

- The PyPI exact-version JSON, file URLs, and JSON Simple API exposed `0.1.12`
  before one HTML Simple API response used by the resolver included the new
  files. A direct PyPI file install smoke passed during that window, and the
  normal resolver install by exact version passed after the simple-index cache
  refreshed.
- `v0.1.12` is a read-only package snapshot for `data-2026.06.11`. It does not
  permit unattended event publication and does not change the policy that
  generated candidates require source-owner review before promotion.
