# Data License

Normalized factual event records, generated feeds, generated indexes, release
manifests, and event fixtures are dedicated to the public domain under CC0-1.0.

APW data includes factual fields such as provider identifiers, dates, source
URLs, checksums, event kinds, short reviewed summaries, and impact
classifications. It does not grant rights to provider pages, provider prose,
screenshots, logos, or trademarks.

Even though CC0 does not require attribution, APW event records keep source URLs,
retrieval timestamps, source authority, and content hashes because provenance is
part of the product contract.
