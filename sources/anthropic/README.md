# Anthropic Sources

Anthropic pricing, status, news, and Claude Platform release-note sources.
Pricing and release notes currently resolve under `platform.claude.com`; status
currently resolves under `status.claude.com`. Legacy Anthropic hosts remain in
`allowed_domains` only for official redirects.

Synthetic parser fixtures cover bounded Claude pricing/model signals,
Statuspage-style incident link hashing, news index links, and dated release-note
sections without copying provider prose. Release-note observations emit
review-only candidates from dates, subjects, source-page evidence, and hashes;
they do not publish ProviderEvents without maintainer review.
