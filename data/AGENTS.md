# Data Agent Guide

- `data/events/` contains reviewed canonical event records.
- `data/candidates/` contains review-only finding candidates and must not be
  treated as published provider facts.
- `data/feeds/`, `data/indexes/`, and `data/releases/` are generated by
  `apw index`.
- Do not hand-edit generated artifacts except to diagnose a failing generator.
- Event data and generated feeds use CC0-1.0.
